# Results Gallery - Image Organization

This folder contains before/after transformation pictures organized by treatment category.

## Folder Structure

### Body Contouring (`body-contouring/`)
- **butt-lifting/** - Butt lifting transformation pictures
- **body-slimming/** - Body slimming and inch loss results
- **body-sculpting/** - Body sculpting and contouring results
- **cellulite-reduction/** - Cellulite reduction treatment results

### Facials (`facials/`)
- **cryotherapy/** - Cryotherapy facial results
- **microdermabrasion/** - Microdermabrasion facial results
- **deep-cleansing/** - Deep cleansing facial results

### Anti-Aging (`anti-aging/`)
- **face-lifting/** - Cryo face & eye lifting results
- **neck-chin/** - RF chin toning and neck treatments

### Specialized Treatments (`specialized/`)
- **back-treatments/** - Back facial and treatment results

## Image Guidelines

- Each image should already contain **before and after side-by-side** in one picture
- Name your images descriptively: `butt-lifting-1.jpg`, `butt-lifting-2.jpg`, etc.
- Supported formats: JPG, JPEG, PNG, WebP
- Recommended size: 1200px wide or larger for best quality
- Keep file sizes reasonable (under 500KB when possible)

## How to Add Images

1. Place your combined before/after images in the appropriate subfolder
2. Update `/data/results.ts` with the image path
3. Example path: `/images/results/body-contouring/butt-lifting/butt-lifting-1.jpg`

The images will automatically appear in:
- The Results Gallery page (with category filters)
- Individual service pages
- Featured results on the home page
