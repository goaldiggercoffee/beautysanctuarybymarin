## IMAGE SOURCING RECOMMENDATIONS

### For Product Images:
- Use professional product photography
- Consider using manufacturer-provided images if available
- Maintain consistent styling across all products
- Ensure proper lighting to show product details

### For Results Images:
- Use real client photos with proper consent
- Maintain client privacy (crop faces if needed)
- Use consistent lighting for before/after pairs
- Take photos from the same angle and distance
- Consider using professional medical photography services

### Stock Images (Temporary):
If real images are not available yet, you can use:
- Unsplash.com for product photography inspiration
- Pexels.com for beauty treatment photos
- Ensure placeholder images maintain the professional aesthetic

---

## PLACEHOLDER BEHAVIOR

The website is designed to handle missing images gracefully:
- **Store**: Shows "Image Coming Soon" with an icon
- **Results**: Shows "Image Coming Soon" message
- All images use Next.js Image component for optimization
- Gradient backgrounds provide visual interest while loading

---

## OPTIMIZATION NOTES

- All images are automatically optimized by Next.js
- Images are lazy-loaded for better performance
- Responsive sizes are generated automatically
- WebP format is served when supported

---

## UPDATING IMAGES

To add new images:

1. Save images with the exact filenames listed above
2. Place them in the appropriate directory
3. The website will automatically display them
4. No code changes needed - just add the image files

For additional products or results:
1. Add new entries to `data/products.ts` or `data/results.ts`
2. Follow the naming convention for new images
3. Update this documentation with new image requirements
